## 📚 О проекте

Этот репозиторий содержит **лабораторные работы** по курсу **Языки программирования-1 (ЯП-1)** на языке **C++**. Каждая работа реализована с учётом таких практик программирования как:

-  **Модульная структура** — заголовочные файлы `.h` и реализации `.cpp` разделены
-  **Человеческие комментарии** — только там, где нужно, без заумностей
-  **Проверки от дурака** — валидация ввода, проверка файлов, защита от ошибок
-  **Интерактивное меню** — удобный пользовательский интерфейс
-  **Фирменный юмор** — "ошибочка(", "проверька", "плак-плак" 😜
